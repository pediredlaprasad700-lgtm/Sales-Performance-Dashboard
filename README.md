# 📊 Sales Performance Dashboard (Tableau)

## 📌 Project Overview

This project presents a Sales Performance Dashboard built using Tableau.

The dashboard includes KPI cards and visualizations to analyze sales performance, profit trends, and regional performance.

---

## 📊 Dashboard Features

* Total Sales KPI
* Total Profit KPI
* Total Quantity KPI
* Monthly Sales Trend
* Sales by Category
* Sales by Region
* Profit by Category
* Interactive Filters

---

## 🛠 Tools Used

* Tableau Public
* CSV Dataset
* Data Visualization

---

## 🔗 Live Dashboard

## 📈 Key Insights

* Technology category shows strong profit.
* West region has higher sales performance.
* Monthly trends help identify sales patterns.
